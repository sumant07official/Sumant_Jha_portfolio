# 🚀 Sumant Jha — Personal Portfolio

A modern, responsive personal portfolio website built with pure HTML, CSS, and JavaScript. No frameworks, no dependencies — just one single file that works everywhere.

## ✨ Features

- **Single file** — entire portfolio in one `HTML` file, no build tools needed
- **Fully offline** — all images, icons, and logos embedded as base64 (no CDN required)
- **Responsive** — works on mobile, tablet, and desktop
- **Dark theme** — deep dark background with purple glow aesthetic
- **Smooth scrolling** — active nav highlight as you scroll
- **Typing animation** — rotating role titles in the hero section
- **Contact form** — opens email client with pre-filled message to your Gmail
- **Real skill logos** — all tech logos embedded as inline SVGs

## 🚀 Deployment

### Option 1 — Vercel (Recommended)
1. Go to [vercel.com](https://vercel.com) and sign up
2. Click **Add New Project**
3. Upload `sumant_jha_portfolio.html`
4. Click **Deploy** → get a live link instantly

### Option 2 — GitHub Pages
1. Create a new GitHub repo
2. Upload `sumant_jha_portfolio.html` and rename it to `index.html`
3. Go to repo **Settings → Pages**
4. Set source to `main` branch
5. Your site is live at `https://yourusername.github.io/repo-name`

### Option 3 — Netlify
1. Go to [netlify.com](https://netlify.com)
2. Drag and drop `sumant_jha_portfolio.html` onto the deploy area
3. Live in seconds ✅
